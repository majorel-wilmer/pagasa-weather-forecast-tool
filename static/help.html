Help & Navigation

What this tool is

This is a decision-support dashboard for WFM/Ops, showing a 5-day rain outlook for each monitored site. Forecast data, rain timing, and intensity are computed from Open-Meteo's hourly model. Tropical cyclone/typhoon tracking on the Live Storm Map still comes from PAGASA's official bulletin, since Open-Meteo doesn't publish cyclone advisories.

Tab-by-tab guide

▦ Dashboard 5-Day Forecast

The main table: one row per site, one column per day. Each cell shows the rain narrative (e.g. "Partly cloudy skies with a high chance of light rain from 7 AM to 11 PM"), the rain-chance pill, and a colored top border for severity. The panel above the table also shows a Model Outlook summarizing the most notable rain/wind signals across all sites for the next 5 days.

◉ Live Storm Map Storm Monitor

Shows PAGASA's official daily bulletin and any active tropical cyclone information, plus a live map from PANaHON. Use this tab specifically for named-storm tracking — the 5-Day Forecast tab does not cover cyclone advisories.

☷ Guidance Detailed Guidance

An expandable, per-site breakdown of each day's forecast with recommended actions (e.g. "check flood-prone routes," "monitor gust forecasts"). Useful when you need more operational detail than the compact dashboard cells show.

◷ Historical Historical Weather

Compares the current month against the same month in the past 5 years for a chosen site, using Open-Meteo's historical reanalysis. Useful for spotting whether a season is running wetter or drier than usual.

Reading a forecast cell

  • Narrative sentence — plain-language description of sky condition, chance of rain, intensity, and the specific rain window (e.g. "from 2 PM to 5 PM") when the model can pin down a window; otherwise "throughout the day."
  • Rain % pill — the daily peak chance of measurable rain.
  • Top border color — the severity tier for that day (see legend below the forecast table on the Dashboard tab).
  • Base badge — the model-computed severity before any admin override is applied.
  • "Display: X" badge — only appears when an admin override changes what's shown; it tells you the displayed severity differs from the model's own calculation.
  • Regional hazard overlay box — appears when the model flags heavy rain, thunderstorm risk, or strong wind gusts for that day.

Severity color coding

  • GREEN — Light rain (<2.5 mm/hr modeled)
  • YELLOW — Moderate rain (2.5–7.5 mm/hr modeled)
  • ORANGE — Heavy rain (>7.5 mm/hr modeled) or thunderstorm risk
  • RED — Admin override; discretionary escalation applied by an operations lead
  • GRAY — No rain classification (dry/clear day)

How to use Admin mode

  1. Click the Admin button in the top-right of the Dashboard tab.
  2. Enter the admin password and click Sign in. (This password is set by whoever deployed the tool — ask your team lead if you don't have it.)
  3. Once signed in, every forecast cell shows three small buttons in its top-right corner: Set YELLOW, Set ORANGE, and Set RED.
  4. Click one to escalate that specific day's displayed severity, regardless of what the model itself calculated — useful when your team has other information (e.g. a local advisory) that the model hasn't caught yet.
  5. Only one override level can be active per cell at a time; clicking a different level replaces the previous one.
  6. Click the same button again (it will say Clear) to remove the override and go back to the model's own severity.
  7. Click Admin again to sign out. Your session stays active for 8 hours or until you sign out.

Overrides only change the displayed severity and color — they don't change the underlying Open-Meteo forecast data, and they're visible to everyone viewing the dashboard.

Other things you can do

  • Export Excel — downloads the same 5-day table as a formatted spreadsheet, with the same severity colors and narrative sentences, plus a Source tab explaining the color legend.
  • Refresh — manually pulls the latest model run instead of waiting for the automatic 30-minute refresh.
  • Dark mode toggle (☾) — switches the dashboard's color theme.
  • Chat assistant — the chat bubble in the bottom-right corner of the Dashboard tab can answer quick questions like "will it rain in Baguio on Thursday?" using the same forecast data.
